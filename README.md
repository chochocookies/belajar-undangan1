# 💌 Belajar Undangan 1 - Fulan & Fulana Wedding

Website undangan digital pernikahan berbasis web yang responsif dan interaktif. Proyek ini dibuat untuk mempelajari dasar-dasar pengembangan web front-end.

---

## 🚀 Fitur Utama
* **Desain Responsif**: Tampilan optimal saat dibuka di HP, tablet, maupun komputer.
* **Informasi Acara**: Detail waktu, tanggal, dan lokasi pernikahan Fulan & Fulana.
* **Hitung Mundur**: Fitur *countdown timer* interaktif menuju hari H acara.
* **Integrasi Peta**: Tautan langsung ke Google Maps lokasi acara untuk memudahkan tamu.

## 🛠️ Teknologi yang Digunakan
* **HTML5** (49.3%) - Menyusun struktur utama halaman web.
* **CSS3** (46.0%) - Mengatur gaya visual, tata letak, dan responsivitas.
* **JavaScript** (4.7%) - Menambahkan logika interaktif seperti jam hitung mundur.

## 💻 Cara Menjalankan Proyek

Jika ingin menjalankan atau melihat proyek ini di komputer lokal Anda:

1. **Clone atau Unduh Repositori**
   ```bash
   git clone https://github.com/chochocookies/belajar-undangan1
   ```
2. **Masuk ke Folder Proyek**
   ```bash
   cd belajar-undangan1
   ```
3. **Buka di Browser**
   * Klik dua kali pada file `index.html` di komputer Anda, atau gunakan ekstensi *Live Server* di VS Code.
